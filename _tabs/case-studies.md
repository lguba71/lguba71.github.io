## Rotary Table Assembly Machine

High-volume automated assembly concept with rotary indexing table and vibratory feeder input.

### Project context

The machine was designed for assembling two components in a high-volume manufacturing environment. The system used vibratory feeder-based material input and a rotary indexing table to organize the assembly process into multiple stations.

### Key data

- Assembly of two components
- Vibratory feeder material input
- Cycle time: 2.5 s
- Annual production volume: approx. 10 million parts/year
- Operator ratio: 1/1
- Autonomy: approx. 45 min

### Engineering relevance

- rotary indexing table concept
- automated assembly process
- station-based machine layout
- feeder integration
- mechanical modules for positioning, assembly, and handling
- design for repeatability and production reliability

### Deliverables

- 3D CAD model
- machine concept
- mechanical station design
- production-oriented design documentation
- design support for manufacturing and assembly

## Safety Leak Test Station

Manual / semi-automatic leak testing station designed for safe and flexible adapter handling.

### Project context

The station was designed to receive multiple leak test adapters without requiring pneumatic connection of the adapters. The concept supported both right-handed and left-handed operators and included safety-related functions to prevent incorrect operation.

### Key functions

- compatible with multiple leak test adapters
- no pneumatic connection required for adapters
- usable by right-handed and left-handed operators
- operation allowed only with adapter installed
- rejected part fixation
- safety-oriented leak testing workflow

### Engineering relevance

- operator-oriented fixture design
- safety-focused mechanical concept
- adapter handling simplification
- ergonomic consideration
- reduction of incorrect operation risk
- fixture and station design for production use

### Deliverables

- fixture / station CAD model
- mechanical concept
- production drawings
- design documentation
- DFM and usability considerations

</details>

## Shiva Rotary Table Semi-Automatic Machine

Semi-automatic rotary table assembly machine for five-component assembly.

The machine concept supported the assembly of five components using vibratory feeder-based part feeding and operator pre-assembly.

### Key data

- vibratory feeder material input
- operator pre-assembly
- cycle time: 6.2 s
- annual production volume: approx. 4.5 million parts/year
- operator ratio: 1/1
- autonomy: approx. 45 min
- project phase: design phase

### Engineering relevance

- semi-automatic assembly concept
- rotary table layout
- feeder integration
- operator-machine interaction
- production-oriented mechanical concept
- assembly process planning

### Deliverables

- machine concept
- 3D CAD model
- station layout
- mechanical design documentation
